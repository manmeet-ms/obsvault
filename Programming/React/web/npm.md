For Installation: [nvm-sh/nvm: Node Version Manager - POSIX-compliant bash script to manage multiple active node.js versions (github.com)](https://github.com/nvm-sh/nvm).
## Initialization

```sh
npm init    # initializes by asking questions to fill up in the .json file

npm init -y # creates a default .json file w/o asking any questions
```

## Installing dev dependencies

```sh
npm install -D <packages> # or --save-dev <packages>

# alternatively
npm i -D <packages>
```
