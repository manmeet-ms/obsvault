> The **`<dialog>`** HTML element represents a dialog box or other interactive component, such as a dismissible alert, inspector, or sub window.

> [!warning]
> Must not use `tabindex` attribute

**Attribute**: `open` - indicates that the dialog is open and can be interacted with

See the docs on [MDN](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/dialog) for detailed information, code examples and *browser compatibility*.
