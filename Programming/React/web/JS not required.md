**See also [UnsuckJS](https://unsuckjs.com/)** (Progressively enhance HTML with lightweight JavaScript libraries - no build tools, no compilers).

- HTML and CSS are preferred over JavaScript for better performance and accessibility.
- CSS can be used for custom styling of form elements using the `appearance` property.
- The `:checked` pseudo-class allows for creating custom switches using checkboxes without JavaScript.
- The `:focus-visible` pseudo-class is useful for highlighting elements for keyboard users.
- Native browser features like `datalist` can be used for autosuggest instead of third-party libraries.
- The native color picker in browsers provides a simple way to select colors.
- Accordion functionality can be implemented using the `<details>` and `<summary>` elements provided by browsers.
- The `open` attribute can be used to set an accordion item as open by default.
- `<dialog>` for modals
