### Story of JavaScript: From Browser to Standalone Executor

Created by Brendan Eich in 1995 while he was an engineer at Netscape Communications Corporation. Initially named Mocha, then LiveScript, it was eventually renamed to JavaScript to capitalize on the popularity of Java, despite having little relation to it.

### JavaScript Evolution & Expansion

**Browser Evolution:**

- **Origin:** Created in 1995 to enhance web interactivity via client-side scripting.
- **Standardization:** Became ECMAScript in 1997, improving compatibility and adoption.
- **Web Development Advancements:** Technologies like AJAX and frameworks (jQuery, AngularJS, React) made JavaScript core to developing sophisticated web applications.

**Standalone Execution Shift:**

- **Node.js Introduction:** Launched in 2009 by Ryan Dahl, enabling server-side JavaScript execution.
- **Cross-Platform Development:** Tools such as Electron and Cordova extended JavaScript to desktop and mobile app development.
- **IoT Integration:** Projects like Johnny-Five brought JavaScript into embedded systems and IoT, showcasing its broad applicability.

**Conclusion:** JavaScript evolved from a browser-based scripting language to a versatile platform for various computing environments, underscoring its flexibility and the vibrant community driving its innovation.
