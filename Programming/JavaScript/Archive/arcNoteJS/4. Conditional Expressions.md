We have 3 forms:

##### Syntax 
```js
if (condition){
    // Code
}
else{
    // Code
}
```

# `if-else if` ladder
```js
if (condition){
    // Code
}
else if (condition){
    // Code
}
// Many else-if(s)...
else{
    // Code
}
```

# Nested `if-else`
```js
if (condition){
    if (condition){
	    if (condition){
		    // Code
		}
    // Code
	}
}
else if (condition){
	if (condition){
	    // Code
	}
    // Code
}
else{
    // Code
}
```

# Switch case 
```js
switch (expression) {
    case value:
        break;
    case 'value2':
        break;
	// and so on...
    default:
        break;
}
```
We use `break` so that it doesn't execute all the cases under it (if any) including default.

# Ternary operator
```js
output = condition ? "Statement of true" : "else statement of False"
```

##### Example
```js
condition = a>7
let output = condition ? "Statement of true" : "else statement of False"
console.log(output);
```
